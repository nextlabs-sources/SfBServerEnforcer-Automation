This is a common DLL project and provide the dependence common tools for SFB.
The DLL must no reference to special depends, like UCMA environment.
Now the common functions include:
1. XML helper
2. JSON helper
3. SQL(Database) helper
4. File operation helper
5. Power Shell Helper
6. SFB Command helper (SFB Server/Client side command analysis)
7. Debug helper


About SFB object info:
1. Current SFB object is: Persistent Chat Category, Persistant Chat Room, Meeting
2. Common module provide classes to manager their informations
3. Control pannel, query policy module need used them
